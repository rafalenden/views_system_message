1. Unpack and move directory views_message to your modules directory.
2. Enable it in the modules list of your site.
3. Create new view via "views" module.
4. Add new header or footer.
	- "Global Message" will appear in the list. 
	- Check and add this item to display.
	- After, settings of this one will appear.
	- Add message text and type.
	- Submit changes.
5. Message will appear in the preview section.
